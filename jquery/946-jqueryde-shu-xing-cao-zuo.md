## jquery的属性操作
对象：有它自己的属性和方法，我们先研究一下jquery的属性操作

### attr
概念：设置属性值或者 返回被选元素的属性值

```javascript	
//获取值：attr()设置一个属性值的时候 只是获取值
		var id = $('div').attr('id')
		console.log(id)
		var cla = $('div').attr('class')
		console.log(cla)
		
```
